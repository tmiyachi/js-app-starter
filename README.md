# js-app-starter

JavsScript を用いた web アプリ開発のための個人用スターター

## Description

- webpack によるバンドル
- ESlint，stylelint による静的コード解析
- Prettier によるフォーマット
- Babel によるトランスコンパイル

詳細は https://www.notion.so/JS-Web-8964358b29714a68898c13399e98435e

## Usage

Install package

```
npm install
```

build

```
npm run build
```

dev

```
npm start
```
